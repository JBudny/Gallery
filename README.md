# Grid-gallery

Image gallery with CSS Grid, flexbox and JS

## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Dependencies nstallation

```
npm install
```

### Running

Type in cmd:

```
node index.js
```

Address and port of the started server:

```
localhost:3000
```
